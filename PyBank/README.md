```bash

- Total Made
- Average Changes
- Most Profit Month and the Figure
- Biggest Loss Month and the Figure
```

### Program Input
```bash
Program Retrieves Data From CSV File (Budget_Data.csv)
```

### Program Output
```bash
- Screen Output
- Text File Output (Financial_Analysis.txt)
```
